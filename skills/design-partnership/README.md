# Skill: Design Partnership Interview Prep

Generates tailored prep materials for PM/design collaboration interviews, runs mock Q&A with structured feedback, and gives an honest readiness assessment.

Built for AI PMs. Includes focused treatment of:
- The value of design when PMs can generate prototypes with AI tools
- The PM's value when everyone can build
- Designing for both humans and agents
- How to demonstrate genuine (not transactional) design partnership

## How to Use

In Claude Code, say:
> `Prep me for [Company] design partnership interview`

Claude will read your candidate profile and company research folder, then ask which phase you want:
1. **Prep Doc** — generates and saves a full prep document
2. **Mock Q&A** — asks questions one at a time, gives feedback on each answer, saves polished answers to your prep doc
3. **Readiness Assessment** — honest rating with specific gaps to close

## Requirements

- `candidate-profile/profile.md` filled in
- `Companies/[CompanyName]/` folder with job description and any research
- Recruiter guidance added to the company folder if available (makes prep significantly more targeted)
